13. На вход подается строка, содержащая несколько предложений. Напишите программу, которая подсчитывает количество предложений в строке. (Предложение содержит только точки) Входные: "Hello. How are you. I'm fine." → Результат: 3 Входные: "This is one sentence." → Результат: 1 Входные: "What's your name. My name is John." → Результат: 2
